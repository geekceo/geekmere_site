Бета-тест предназначен для выявления багов и недоработок в промежуточной версии, когда Вы - пользователи - можете помочь разработчикам в производстве конечного продукта. Тем, кто помог в вычислении недработок мы вышлем индивидуальные копии продукта до основого релиза.

Список продуктов для которых открыт бета-тест

EVA-8 Console

EVA-8 Console предназначена для гиков и любителей терминалов Linux и Windows.



Правила теста:

• Не давать на вход пути с пробелами в названии директорий и файлов

• Работает пока только с изображениями

• Не совместим с шифром прошлых версий, поэтому старый код не воспримет

• После запуска написать в строке ввода команд команду "help" для получения списка команд

• НЕ удалять бинарные файлы ни из одной папки

• При нахождении бага, сообщить о нем при помощи формы внизу страницы
